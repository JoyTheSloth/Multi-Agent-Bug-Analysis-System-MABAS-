

# 🎯 What You Need To Finish (Backend Logic Checklist)

---

## ✅ 1. Connect UI → Backend (VERY IMPORTANT)

Your UI should:

* Send:

  * bug report
  * logs

👉 To backend (Python API / function)

Example flow:

```text
UI → backend → agents → tools → output → UI
```

---

## ✅ 2. Implement Agents (Core Logic)

Make sure these actually run (not static):

* Triage Agent
* Log Analyst Agent
* Reproduction Agent ⚠️ (MOST IMPORTANT)
* Fix Planner Agent
* Reviewer Agent

👉 Each should return real output

---

## ✅ 3. Tool Execution (MANDATORY)

At least 2 tools must be real:

* parse logs
* extract stack trace
* run repro script

👉 Example:

```python
def run_repro_script(path):
    import subprocess
    return subprocess.run(["python", path], capture_output=True, text=True)
```

---

## ⚠️ 4. REPRO SCRIPT (THIS DECIDES YOUR SELECTION)

Your system must:

* Generate `repro.py`
* Run it
* It MUST fail

int("abc")
